Conversion

Module for unit conversions.

Binint: Convert binary to an integer.

Usage: -binint <binary>
Output:
    Converts the input binary to its integer representation.

Binstr: Convert binary to a string

Usage: -binstr <binary>
Output:
    Converts the input binary to its string representation.

Cm: Convert centimeters to feet and inches.

Usage: -cm
Alias: -centimeters
Output:
    Turns cm (height) to ft + inch
    (metric to imperial) by approximation

Dechex: Convert decimal into hex.

Usage: -dechex <hex>

Decode: Decode from b32, b64, b85, rot13, hex.

Usage: -decode <method>
Methods:
    b32
    b64
    b85
    rot13
    hex

Encode: Encode to: b32, b64, b85, rot13, hex.

Usage: -encode <method>
Methods:
    b32
    b64
    b85
    rot13
    hex

Ft: Convert feet.inches to centimeters

Usage: -ft <value>
Alias: -feet
Output:
    Turns height ft.inch (Eg: 5.11)
    to cm (imperial to metric)

Hexdec: Convert hex to decimal.

Usage: -hexdec <hex>

Intbin: Convert an integer to binary.

Usage: -intbin <integer>
Output:
    Converts the input integer to its binary representation.

Kg: Convert kilograms to pounds.

Usage: -kg <value>
Alias: -kgs, kilograms
Output: Turn kg into lb (metric to imperial)

Lb: Convert pounds to kilograms

Usage: -lb <value>
Aliases: -lbs, pounds, pound
Output: Turn lb into kg (imperial to metric)

Morse: Converts ascii to morse code.

Usage: -morse <content>
Output: Converts ascii to morse code.
Notes:
    Accepts a-z and 0-9.
    Each letter is comprised of "-" or "."
    and separated by 1 space.
    Each word is separated by 4 spaces.

Morsetable: Show the morse lookup table

Usage: -morsetable
Output:
    Prints out the morse code lookup table.

Strbin: Convert a string to binary.

Usage: -strbin <string>
Output:
    Converts the input string to its binary representation.

Unmorse: Converts morse code to ascii.

Usage: -unmorse <morse>
Output: Converts morse code to ascii.
Notes:
    Each letter is comprised of "-" or "."
    and separated by 1 space.
    Each word is separated by 4 spaces.
